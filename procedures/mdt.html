Procédure : Déploiement Windows — MDT / WDS

Objectif

Installer et configurer un serveur MDT/WDS permettant le déploiement automatisé de Windows 10/11 via PXE.

Étapes principales

  1. Installation de Windows Server
  2. Installation du rôle WDS
  3. Installation de MDT + ADK + WinPE
  4. Création du Deployment Share
  5. Importation de l’ISO Windows
  6. Création de la Task Sequence
  7. Configuration de Bootstrap.ini et CustomSettings.ini
  8. Génération de l’image LiteTouchPE
  9. Ajout de l’image dans WDS
  10. Déploiement PXE sur un client
