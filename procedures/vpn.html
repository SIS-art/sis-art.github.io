Procédure : VPN (OpenVPN & WireGuard)

Objectif

Créer un réseau sécurisé via VPN, avec OpenVPN et WireGuard, incluant certificats, routage et tests de connectivité.

Étapes principales

  1. Installation du serveur VPN
  2. Configuration réseau (ports, firewall, IP fixe)
  3. OpenVPN : génération des certificats
  4. OpenVPN : configuration serveur
  5. OpenVPN : configuration client
  6. WireGuard : génération des clés
  7. WireGuard : configuration serveur
  8. WireGuard : configuration client
  9. Routage & NAT
  10. Tests de connectivité
