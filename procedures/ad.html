Procédure : Active Directory (AD DS)

Objectif

Installer, configurer et vérifier un domaine Active Directory complet avec DNS et DHCP.

Étapes principales

  1. Installation de Windows Server
  2. Configuration réseau (IP fixe, DNS local)
  3. Installation du rôle AD DS
  4. Promotion du serveur en contrôleur de domaine
  5. Installation et configuration DNS
  6. Installation et configuration DHCP
  7. Création des OU
  8. Création des utilisateurs
  9. Création et application des GPO
