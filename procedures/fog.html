Procédure : FOG Project

Objectif

Installer et configurer un serveur FOG sous Debian pour capturer et déployer des images Windows via PXE.

Étapes principales

  1. Installation de Debian
  2. Configuration réseau (IP fixe, DNS, passerelle)
  3. Installation des dépendances FOG
  4. Installation du serveur FOG
  5. Configuration de l’interface web FOG
  6. Enregistrement du client dans FOG
  7. Création d’une image
  8. Capture de l’image via PXE
  9. Déploiement de l’image sur un autre poste
